## FTE計算
- 常勤 = 1.0 FTE
- 非常勤 = 0.5 FTE（デフォルト、変更可能）
- Dr FTE = 常勤Dr + 非常勤Dr × 0.5
- DH FTE = 常勤DH + 非常勤DH × 0.5

# 7. 配賦ロジック仕様

## 7.1 三層コスト構造

### 直接原価（COGS）
部門に直接紐づくコスト。部門別に直接計上。
- 技工料
- 直接材料費
- 外注費
- 診療区分直接費

### 直接計上費
特定部門に帰属が明確なコスト。配賦不要。
- 特定部門専任人件費
- 特定施策専用広告費
- 自費部門専用コスト

### 間接費
部門横断的に発生し、配賦が必要なコスト。
- 受付人件費、共通スタッフ人件費
- 家賃、光熱費、通信費
- リース料、システム費
- 広告費（共通分）、消耗品費
- 修繕費、減価償却費、雑費

## 7.2 配賦計算フロー

```
1. 費目ごとに配賦対象ON/OFFを確認
2. 配賦対象の費目について:
   a. コストドライバーを取得
   b. 各配賦先（部門）のドライバー量を取得
   c. 総ドライバー量を計算
   d. 総ドライバー量 = 0 → 未配賦扱い
   e. コストドライバー・レート = 費目金額 ÷ 総ドライバー量
   f. 各部門の配賦額 = 部門ドライバー量 × レート
   g. 手動調整額があれば加算
3. 部門別採算を計算:
   売上 - 直接原価 = 粗利
   粗利 - 直接計上費 = 配賦前利益
   配賦前利益 - 間接費配賦額 = 配賦後営業利益
```
